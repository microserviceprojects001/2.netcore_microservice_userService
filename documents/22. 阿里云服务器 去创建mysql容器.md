# 方案
```
-- 撤销之前的错误授权
REVOKE ALL PRIVILEGES ON _._ FROM 'abel'@'%';

-- 授予正确权限（使用星号通配符）
GRANT ALL PRIVILEGES ON *.* TO 'abel'@'%' WITH GRANT OPTION;

FLUSH PRIVILEGES;
```
